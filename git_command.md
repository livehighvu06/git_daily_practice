#Git 指令筆記

- `git init`
  在 folder 裡建立 Git 初始化

- `git status`
  查看 git 狀況

- `git add .`
  加入所有檔案

- `git add 檔案名稱.副檔名`
  加入指定檔案

- `git commit`
  提交目前的異動

- `git commit -m "異動說明內容"`
  提交目前的異動並設定說明

- `git log`
  查看之前的 commit 紀錄

- `git push`
  將本地端 Repo 的 commit 發佈到遠端

- `git push -u origin [BRANCH_NAME]`
  發佈到遠端指定的分支

- `git branch`
  查看分支

- `git branch [BRANCH_NAME]`
  建立分支

- `git checkout [BRANCH_NAME]`
  切換到指定的分支

- `git checkout -b [BRANCH_NAME]`
  建立並切換到指定分支
